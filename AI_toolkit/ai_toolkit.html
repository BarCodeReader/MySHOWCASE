Background Removal

How it works

By applying Saliency Object Detection model, we can obtain the prediction mask of that saliency object.

A threshold is applied to the mask in order to further improve the prediction accuracy

Applying the mask on original image, we can extract the object and achieve backtround removal function

Demonstration

By applying Saliency Object Detection, we can perform Background Removal function on client's image.

Object Removal

How it works

User can specify areas where they want to remove the object, and thus a mask is obtained

Feed the mask and the maksed image into Image Inpainting model, we can achieve Object Removal function

Use can choose to either use Saliency Object as the mask or they manually draw a box on the image

Demonstration

By applying Image Inpainting, we can perform Object Removal function on client's image.




Parallex

How it works

Apply Saliency Object detection for object extraction

Apply Image Inpainting mode to fill the background

By adjusting the movement and scale of the object and background, we can achieve Parallex effect

Demonstration

Bokeh

How it works

By applying Saliency Object Detection model, we can obtain the prediction mask of that saliency object.

A threshold is applied to the mask in order to further improve the prediction accuracy

Background image is then blured by applying pre-defined kernel

Applying the mask on original image and new background, we can achieve Bokeh effect

Animation Recommendation

How it works

By applying Saliency Object and Saliency Map model, we can generate a Hybrid Saliency Map

The hybrid saliency map served as a "foreground-background map with attention"

Based on the hybrid saliency map and pre-defined algorithm, we can generate animation suggestions

Demonstration

By applying Saliency Object and Saliency Map model, we can perform Animation Recommendation function on client's image.

     Recommended Animation: Move Down




Caption Positioning

How it works

By applying Saliency Object and Saliency Map model, we can generate a Hybrid Saliency Map

The hybrid saliency map served as a "foreground-background map with attention"

A neural network called CapNet will take the saliency map as input and generate position score

Positions with top-3 scores will be recommended to user
