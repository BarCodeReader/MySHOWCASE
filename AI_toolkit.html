AI Editing Toolkit

A COMPREHENSIVE AI-BASED IMAGE EDITING TOOLKIT FOR FAST AND EASY IMAGE PROCESSING EXPERIENCE

Image editing tools, like PhotoShop and Illustrator, are necessary for artwork designers. Most of the usage of these tools are for background removal, background change and image mixing. However, thought they are very powerful, they still require a lot of human interaction such as refine the boundary, tune the color or transparency. We want to design an editing toolkit, by using AI, to provide our users a much easier and faster image editing experience, yet with high quality.

Thus we designed an AI model, to extract object user selected from an image and generate alpha masks.

Background Removal

We designed an AI model, which can perform high quality image matting for the object you selected.

Alpha mask will be generated and is used to extract the object



Applications

When there is no selection of objects, the most obvious object will be used for matting.

When user selected the object or part of the object, we can then perform a lot of effects, like change the color.

Background Change

When you can remove the background, you can also change the background.
